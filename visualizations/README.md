# Visualizations

Charts and visualizations based on this dataset are available at:

**[unvow.com/child-support-guidelines](https://unvow.com/child-support-guidelines)**

Includes:
- Guideline model distribution (income shares vs. percentage of income vs. Melson)
- Income threshold comparisons by state
- Base percentage rates for 1–6 children
- Parenting time adjustment thresholds
- Add-on expense allocation methods

All visualizations are built with pure CSS/HTML for fast loading and accessibility.

---

Community-contributed visualizations are welcome. Submit a PR with your chart or notebook and we'll add it here.
